MythTrans-llama31-8B

View on GitHub AI/ML

A specialized language model fine-tuned from LLaMA 3.1 8B, designed specifically for translating English descriptions of mythical beasts into Classical Chinese. This project combines the power of modern language models with traditional Chinese literary style to create authentic translations of mythical creatures.

Key Features

  • Specialized translation of mythical creature descriptions
  • Authentic Classical Chinese output
  • Fine-tuned on LLaMA 3.1 8B model
  • Maintains cultural and literary context
  • High-quality translation preservation

Technical Details

  • Based on LLaMA 3.1 8B architecture
  • Fine-tuned with specialized mythical creature dataset
  • Optimized for Classical Chinese literary style
  • Efficient inference capabilities
  • Comprehensive documentation and usage examples
